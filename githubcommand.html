GitHub Commands - Developer Guide

1. Initial Setup

Configure Git Identity
git config --global user.name "Your Name"
git config --global user.email "your@email.com"
Sets your name and email for all commits on your machine.

2. Repository Basics

Initialize Repository
git init
Creates a new Git repository in the current directory.
Clone Repository
git clone <repository-url>
Downloads a repository from GitHub to your local machine.

3. Staging & Committing

Check Status
git status
Shows modified files and staging area status.
Stage Files
git add <filename>
git add .
Stages specific file or all changes for commit.
Commit Changes
git commit -m "Your commit message"
Creates a snapshot of staged changes with a message.

4. Branches

List Branches
git branch
git branch -a
Lists local branches or all branches (including remote).
Create Branch
git branch <branch-name>
Creates a new branch from current branch.
Switch Branch
git checkout <branch-name>
git switch <branch-name>
Switches to an existing branch.
Create & Switch Branch
git checkout -b <branch-name>
git switch -c <branch-name>
Creates and switches to a new branch in one command.

5. Remote Operations

Add Remote
git remote add origin <repository-url>
Connects your local repo to GitHub repository.
Push Changes
git push origin <branch-name>
Uploads commits to GitHub repository.
Pull Changes
git pull origin <branch-name>
Downloads and merges latest changes from GitHub.

6. Merging

Merge Branch
git merge <branch-name>
Merges specified branch into current branch.

7. History & Diff

View Commit History
git log
git log --oneline
Shows commit history. --oneline shows compact view.
View Changes
git diff
git diff <branch1> <branch2>
Shows differences between working directory or branches.

8. Undo Changes

Undo Staged Changes
git reset <filename>
Removes file from staging area.
Discard Changes
git checkout -- <filename>
Discards changes in working directory.
Undo Last Commit
git revert HEAD
Creates new commit that undoes last commit.
